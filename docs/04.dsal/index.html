Ch4: dsal

Overview

The dsal module in fast_io provides a set of simple, predictable containers. These containers form the foundation for storing and organizing data in your programs.

In this chapter, you will learn:

  • how ::fast_io::string stores and grows character sequences
  • how ::fast_io::vector manages a growable sequence of elements
  • how ::fast_io::array provides a fixed‑size sequence
  • how to build 2D, 3D, and higher‑dimension structures by nesting containers

These four containers give you a solid foundation for working with sequences of data. They also prepare you for Chapter 5, where you will learn how pointers interact with these containers.
