Ch12: random

Overview

This chapter covers random number generation in fast_io. You'll learn about cryptographically secure random sources, the importance of using distributions correctly, and how to work with both secure engines and reproducible PRNGs.

Random number generation is essential for cryptography, simulations, gaming, and many other applications. fast_io provides secure, platform-specific random number sources that integrate seamlessly with its I/O system.
