Ch11: I/O

Overview

This chapter covers fast_io’s I/O system — the core of the library. We will learn how to perform file I/O, define custom types for printing and scanning, use manipulators, understand the operations layer, and work with every level of the OS file API.

Sub-chapters

  • Ch11.1: File I/O — ibuf_file, obuf_file, iobuf_file, u8iobuf_file
  • Ch11.2: Custom Types — print_define, print_reserve_define, print_scatter_define, scan_context_define
  • Ch11.3: Manipulators — hex, oct, bin, boolalpha, alignment, and more
  • Ch11.4: I/O Operations — write_some, write_all, pwrite, scatter_write
  • Ch11.5: Buffered vs Unbuffered I/O — native_file vs iobuf_file
  • Ch11.6: Mutex I/O — thread-safe I/O with lockable file types
  • Ch11.7: Random Access I/O — seek, tell, positional I/O
  • Ch11.8: File Loader — memory-mapped files with native_file_loader and allocation_file_loader
  • Ch11.9: Relationship with String — concat_fast_io and to<T> use the same I/O system
  • Ch11.10: Socket Files — native_socket_file, iobuf_socket_file, u8http_header_buffer
  • Ch11.11: C-Style I/O — FILE*, c_file, c_io_observer, printf safety
  • Ch11.12: C++ iostream — fstream, filebuf_file, filebuf_io_observer
  • Ch11.13: POSIX File APIs — open, openat, posix_file, posix_io_observer
  • Ch11.14: Win32 File APIs — CreateFileW/A, win32_file, win32_io_observer
  • Ch11.15: NT File APIs — NtCreateFile, nt_file, nt_io_observer
  • Ch11.16: Wine File APIs — Wine compatibility layer
  • Ch11.17: Relationship Between File Layers — the full layer hierarchy and interoperability
