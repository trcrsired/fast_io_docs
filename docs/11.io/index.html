Ch11: I/O

Overview

This chapter covers fast_io’s I/O system — the core of the library. We will learn how to perform file I/O, define custom types for printing and scanning, use manipulators, understand the operations layer, and work with every level of the OS file API.

Important assumption: This chapter assumes you are running in a standard hosted environment with full filesystem access. If you are running in an application sandbox (e.g., mobile app sandboxes, browser WASM sandbox, container restrictions) or a freestanding environment (e.g., embedded systems without an OS, kernel development), many of the assumptions about file I/O break down. We will discuss these special environments in future chapters.
