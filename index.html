fast_io

This project is more than documentation for the fast_io library. It is a **complete tutorial for learning C++**, built around fast_io. Because fast_io changes the way C++ is written and understood, we will use the fast_io library as our primary tool to learn C++ — making the language simpler, safer, and more powerful.
